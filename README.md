# Viral Growth Hacks

This repository contains the assets and server for the Viral Growth Toolkit, a productized digital asset bundle for creators and small businesses.

## Structure

- `/assets`: Contains the digital products (`toolkit-basic.zip`, `toolkit-full.zip`).
- `/landing`: Contains the `index.html` landing page and `success.html` post-payment page.
- `/server`: A simple Node.js Express server to handle payments and deliver the digital assets.
- `/marketing`: Contains DM templates, ad copy, and email templates for outreach.

## Getting Started

1.  Navigate to the `server` directory.
2.  Run `npm install` to install dependencies.
3.  Run `npm start` to start the server.

The server will be running on `http://localhost:3000`.
